================================================================================
                     EduBattle — Установка и запуск (Linux)
================================================================================

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Ubuntu 20.04+ / Debian 11+ / Fedora 36+
- Python 3.8+
- ОЗУ: 4 ГБ
- Свободное место: 500 МБ
- Интернет-соединение
- Аккаунт Telegram

--------------------------------------------------------------------------------
УСТАНОВКА
--------------------------------------------------------------------------------

1. Перейдите в папку с файлами:
   cd ~/Downloads/olimp_lin

2. Установите зависимости:
   pip install -r requirements.txt

   Если не работает:
   pip3 install -r requirements.txt

   Или вручную:
   pip3 install flask flask-socketio python-telegram-bot

3. Установите переменную окружения:

   Узнать свой Telegram ID: @funstat или @syndver_bot

   Временно:
   export MAIN_ADMIN_TG_ID=123456789

   Постоянно:
   echo 'export MAIN_ADMIN_TG_ID=123456789' >> ~/.bashrc
   source ~/.bashrc

4. Выдайте права на исполнение:
   chmod +x db_editor launcher.py server.py

5. Запустите:
   python3 launcher.py

--------------------------------------------------------------------------------
ДОСТУП К САЙТУ
--------------------------------------------------------------------------------

Основной сайт:        http://localhost:8080
Расширенная панель:   http://localhost:5000

Тестовый вход:
  Email:    admin@edu.ru
  Пароль:   admin123
  Никнейм:  Admin

--------------------------------------------------------------------------------
РАСШИРЕННАЯ ПАНЕЛЬ (2FA через Telegram)
--------------------------------------------------------------------------------

1. Откройте @syndver_bot в Telegram (с аккаунта суперадмина)
2. Нажмите "Добавить администратора" → получите 4-значный код
3. С другого аккаунта: "Получить ключ входа" → введите 4-значный код
4. Введите email (admin@edu.ru) и никнейм (Admin)
5. Получите 16-значный код → введите на http://localhost:5000

--------------------------------------------------------------------------------
ЧАСТЫЕ ПРОБЛЕМЫ
--------------------------------------------------------------------------------

Порт занят:
  lsof -i :8080
  kill -9 <PID>

Ошибка pip:
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt

Сброс БД: удалите database.db и перезапустите сервер
