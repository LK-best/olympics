================================================================================
                     OlimpPrep — Установка и запуск (Windows)
================================================================================

СИСТЕМНЫЕ ТРЕБОВАНИЯ:
- Windows 10/11 (64-bit)
- ОЗУ: 4 ГБ
- Свободное место: 500 МБ
- Интернет-соединение
- Аккаунт Telegram

--------------------------------------------------------------------------------
УСТАНОВКА
--------------------------------------------------------------------------------

1. Скачайте и распакуйте архив olimp_win.zip

   ВАЖНО: Путь НЕ должен содержать кириллицу!
   ✓ C:\Projects\olimp_win\
   ✗ C:\Мои проекты\олимпиада\

2. Установите переменную окружения MAIN_ADMIN_TG_ID:

   Узнать свой Telegram ID: @funstat или @syndver_bot

   Способ 1 (временно, в CMD):
   set MAIN_ADMIN_TG_ID=123456789

   Способ 2 (постоянно):
   Win+R → sysdm.cpl → Дополнительно → Переменные среды → Создать
   Имя: MAIN_ADMIN_TG_ID
   Значение: ваш_telegram_id

3. Запустите olimp_server.exe

--------------------------------------------------------------------------------
ДОСТУП К САЙТУ
--------------------------------------------------------------------------------

Основной сайт:        http://localhost:8080
Расширенная панель:   http://localhost:5000

Тестовый вход:
  Email:    admin@edu.ru
  Пароль:   admin123
  Никнейм:  Admin

--------------------------------------------------------------------------------
РАСШИРЕННАЯ ПАНЕЛЬ (2FA через Telegram)
--------------------------------------------------------------------------------

1. Откройте @syndver_bot в Telegram (с аккаунта суперадмина)
2. Нажмите "Добавить администратора" → получите 4-значный код
3. С другого аккаунта: "Получить ключ входа" → введите 4-значный код
4. Введите email (admin@edu.ru) и никнейм (Admin)
5. Получите 16-значный код → введите на http://localhost:5000

--------------------------------------------------------------------------------
ЧАСТЫЕ ПРОБЛЕМЫ
--------------------------------------------------------------------------------

Порт занят:
  netstat -ano | findstr :8080
  taskkill /PID <номер> /F

Сброс БД: удалите database.db и перезапустите сервер
